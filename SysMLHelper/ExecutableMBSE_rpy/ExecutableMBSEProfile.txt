This is the ExecutableMBSE profile component of the SysMLHelper profile (V3.0.x.Rhp831.PreRelease). It provides an accelerated method support for executable MBSE with a focus on guard-based state machine simulation and high-level activity diagram based use case modeling which places a focus on the creation of textual requirements from use case steps. The core method supported is called ROCKETS (Requirements Originating in the Context of Executable Transitions on Statecharts), www.executablembse.com.
